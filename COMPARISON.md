## New Command: `devutils utils` - ALL Utilities in ONE Place

```bash
# Date/Time
devutils utils now              # 2024-01-15 10:30:45 UTC
devutils utils ts              # Unix timestamp
devutils utils uuid           # Generate UUID

# Passwords
devutils utils pass           # Random 16-char password
devutils utils pass 32        # 32-char password

# Strings
devutils utils rev "hello"    # Reverse: olleh
devutils utils len "hello"    # Length: 5
devutils utils lines "a\nb"   # Line count
devutils utils words "hello world"  # Word count
devutils utils wc "hello world" # lines words chars

# Case
devutils utils upcase "hello" # HELLO
devutils utils downcase "HELLO" # hello
devutils utils trim "  hi  "  # hi

# Encoding
devutils utils hex "hello"   # 68 65 6c 6c 6f
devutils utils unhex "68 65"  # he

# JSON
devutils utils jsonfmt '{"a":1}' # Pretty print

# Git
devutils utils gitlog        # Recent commits
devutils utils gitdiff       # Diff stats
devutils utils gitbranch     # List branches
devutils utils gitstatus     # Short status
```
