## Commands to Demo

```bash
# FAST file search
./devutils find "main"

# AI semantic search (THE KILLER FEATURE)  
./devutils search "login" --semantic

# Explain code
./devutils explain "def add(a,b): return a + b"

# Generate code
./devutils generate "REST API user CRUD"

# Debug help
./devutils debug "for i in range(10): print(i"

# System info
./devutils system

# Git status
./devutils status
```
